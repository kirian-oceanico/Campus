Solicitud de datos personales para la certificación

Para obtener la certificación asociada al programa formativo, es imprescindible que los datos personales que proporciones a Océano Medicina sean correctos.

Si no es usted la persona que va a realizar el programa, debe ceder la licencia al usuario final, el cual deberá registrarse en el sistema con sus datos personales.
